# Niome

Personal wayland compositor and shell.
It combines features from gnome and tiling window managers with my preferences.

## Features

- Floating Windows
- Dynamic Tiling
- Dynamic Workspaces
- Animations
- Rounded Windows
- Window Borders
- Tabbed Window Groups

## Shortcuts

`Meta`: Open Overviewy

`Meta + Drag`: Move/Resize Window

`Meta + Up`: Maximise Window

`Meta + Down`: Demaximise/Float Window

`Meta + Left`: Tile Window in left half

`Meta + Down`: Tile Window in right half

`Meta + Q`: Close Window

`Meta + T`: New Window in Window Group

`Meta + V`: Open Clipboard History

`Meta + .`: Open Character, Emoji, GIF, etc. Selector

## Gestures

`Three Finger + Move Vertical`: Open/Close Overview

`Three Finger + Move Horizontal`: Scroll through
